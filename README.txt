Using API you can able to display beer info and beer and review in the page.

== Description ==

Plugin will help to display beer info and latest beer review and rating in the web page.

Page will be create automatically when plugin active so you just need to check page in the front-end side.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `garrison_brewing.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('beer-info'); ?>` in your templates
